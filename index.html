Tetris Retrô: Uma Webpage Personalizada

Um remake do clássico Tetris com visual retrô e jogabilidade fluida, desenvolvido em JavaScript e CSS. O projeto recria a experiência nostálgica do jogo original, com uma interface simples e responsiva que funciona diretamente no navegador.

A estrutura do site é construída com HTML, responsável por definir a semântica e a organização do conteúdo. Já o CSS é utilizado para aplicar O HTML organiza o conteúdo começando por um cabeçalho com o título do site, seguido por uma divisão principal, container que exibe o propósito da página e uma seção flexível (.livros) com a coleção de livros. Cada obra é apresentada em seu próprio contêiner livro, incluindo a imagem da capa, informações detalhadas e um link para compra. O design responsivo é assegurado pelo uso da meta tag viewport e por um layout flexível que se adapta a diferentes tamanhos de tela.los visuais à página, como cores, fontes e a disposição dos elemento na tela.

O CSS personaliza o visual do site, usando variáveis para cores, estilizando o texto com uma fonte importada do Google Fonts e aplicando um esquema de cores suaves e botões interativos. O uso de HTML e CSS é importante pois assim é possível criar um site acessível e esteticamente agradável sem a necessidade de scripts complexos, com foco na usabilidade e na experiência da pessoa usuária. A escolha da tipografia e do esquema de cores contribui para a atmosfera acolhedora do site, incentivando a exploração do jogo.

Mário Retrô: Uma Webpage Personalizada

O projeto "Mario Runner" é um jogo 2D simples desenvolvido para navegadores, inspirado no universo do Mario. Nele, o personagem corre automaticamente por um cenário contínuo, e o jogador deve fazê-lo pular para evitar canos que aparecem como obstáculos. A mecânica principal gira em torno do pulo, acionado por tecla ou toque na tela, e da detecção de colisões que determinam o fim do jogo.

O desenvolvimento foi feito utilizando HTML5 para a estrutura da página e dos elementos do jogo, CSS3 para as animações e estilização do personagem, do cenário e dos obstáculos, e JavaScript puro (vanilla) para implementar a lógica principal. Toda a movimentação do jogo, como o deslocamento dos canos e o controle de pulo, foi criada sem o uso de bibliotecas externas.

Esse projeto me permitiu praticar conceitos importantes como manipulação do DOM em tempo real, uso de animações CSS sincronizadas com eventos JavaScript, além de uma introdução à lógica de game loop e controle de interações com o usuário. Trata-se de uma experiência introdutória ao desenvolvimento de jogos com tecnologias web, com foco em fluidez visual e jogabilidade simples.

Jogo da velha: Uma Webpage Personalizada

O projeto é um jogo da velha interativo e educativo, desenvolvido para proporcionar uma experiência divertida e desafiadora. A aplicação utiliza uma série de lógicas e gráficos simples, associados a diferentes elementos (como os símbolos de "X" e "O") para criar um ambiente de aprendizado envolvente. O objetivo é ensinar conceitos básicos de lógica e estratégia, enquanto os usuários se divertem competindo em partidas de jogo da velha. Além disso, o jogo inclui modos de dificuldade ajustáveis, permitindo que jogadores de diferentes níveis de habilidade possam participar. Através de interações dinâmicas e uma interface amigável, o projeto visa promover o desenvolvimento de raciocínio lógico e tomada de decisões rápidas de maneira divertida e acessível.

Cada ator no projeto possui scripts que determinam seu comportamento em resposta a eventos específicos , como o início do jogo (quando a bandeira é clicada), cliques dos jogadores ou ações realizadas na interface. Esses eventos acionam ações como a atualização do tabuleiro, a troca de símbolos entre os jogadores, a verificação de vitórias ou empates e a exibição de mensagens informativas. Por exemplo, quando um jogador faz uma jogada, o sistema verifica automaticamente se houve uma vitória ou empate, exibindo a mensagem correspondente. Além disso, o jogo pode exibir mensagens de incentivo, alterar o visual do tabuleiro ou até mesmo mudar de nível de dificuldade dependendo das ações do jogador.

A importância das ferramentas utilizadas, como os blocos de código visual do Scratch, está na sua acessibilidade e na capacidade de envolver aprendizes de diversas idades no desenvolvimento de habilidades de lógica e raciocínio estratégico de maneira prática e intuitiva. O uso de elementos dinâmicos, como a atualização do tabuleiro, movimentos dos jogadores e feedback visual, contribui para uma experiência de jogo mais envolvente e interativa. Essa abordagem estimula os usuários a pensarem de forma ativa sobre as jogadas e as consequências de suas ações, promovendo um aprendizado mais significativo e estimulante, onde o usuário se torna protagonista do seu próprio processo de aprendizagem.
